# Commodities Analytics — Mobile

Aplicativo móvel desenvolvido com Expo para acompanhamento ágil de preços e recebimento de alertas críticos.

## Tecnologias Principais

- **Plataforma:** Expo / React Native
- **Roteamento:** Expo Router
- **Ícones:** Lucide React Native
- **Gráficos:** React Native Chart Kit

## Funcionalidades

- **Alertas Push:** Recebimento imediato de alertas de variação de preço.
- **Dashboard Móvel:** Resumo simplificado do mercado na palma da mão.
- **Registro Simplificado:** Fluxo rápido de autenticação e criação de conta.

## Configuração e Instalação

### 1. Instalação

```bash
cd mobile
npm install
```

### 2. Execução

```bash
# Iniciar o servidor Expo
npx expo start
```

### 3. Visualização

- Utilize o aplicativo **Expo Go** em seu dispositivo físico.
- Ou utilize emuladores Android/iOS.
