# Commodities Analytics — Frontend

Interface web interativa para visualização de análises de mercado, dashboards de preços e gestão de compras corporativas.

## Tecnologias Principais

- **Framework:** Next.js
- **Biblioteca UI:** React
- **Gráficos:** Recharts
- **Ícones:** React Icons
- **Linguagem:** TypeScript

## Funcionalidades

- **Dashboards Dinâmicos:** Visualização em tempo real de variações de preços.
- **Histórico de Mercado:** Gráficos interativos para análise de tendências.
- **Gestão de Empresas:** Cadastro e controle de perfis corporativos.
- **Monitoramento de Alertas:** Visualização centralizada de notificações do sistema.

## Configuração e Instalação

### 1. Instalação

```bash
cd frontend
npm install
```

### 2. Execução

```bash
# Desenvolvimento
npm run dev

# Build de Produção
npm run build
npm run start
```

O frontend estará disponível em `http://localhost:3000`.
